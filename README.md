# Copilot Conductor

A CLI tool for defining and running multi-agent workflows.

## Installation

```bash
uvx copilot-conductor
```

## Usage

```bash
conductor run workflow.yaml
```

## Development

This project uses [uv](https://github.com/astral-sh/uv) for dependency management.

```bash
# Install dependencies
uv sync

# Run tests
uv run pytest

# Run linting
uv run ruff check .
```
